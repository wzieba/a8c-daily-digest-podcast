# Audio Digest Script

This script automates the creation of an audio digest from a text summary generated by Claude, and uploads it to Pocket Casts.

## Prerequisites

1.  **Python 3.8+**
2.  **Claude CLI**: Installed and authenticated (`claude login`).
3.  **Pocket Casts Account**: Premium/Plus subscription might be required for "My Files".
4.  **Google AI Studio API Key**: For Gemini API.

## Setup

1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    playwright install chromium
    ```

2.  Create a `.env` file in this directory with your credentials:
    ```env
    GOOGLE_API_KEY=your_google_api_key
    POCKET_CASTS_EMAIL=your_email
    POCKET_CASTS_PASSWORD=your_password
    ```

## Usage

Run the script:
```bash
python digest_to_audio.py
```

The script will:
1.  Run `claude "/context-a8c:digest"` to generate the text.
2.  Convert the text to audio using Google Gemini (`gemini-2.0-flash-exp`).
3.  Upload the resulting `digest.wav` to Pocket Casts.
